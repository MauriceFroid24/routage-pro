# Application de routage Excel - V3

## Lancement simple

Double-clique sur :

`LANCER_APPLI.bat`

Le lanceur crée un environnement local, installe les modules puis ouvre l'application.

## En cas d'erreur pandas / Visual Studio

Cela arrive surtout avec Python 3.14. Installe Python 3.12, coche `Add Python to PATH`, puis relance `LANCER_APPLI.bat`.
